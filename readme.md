# Lentokonepeli
The app is built using Python and Node.js. Make sure you have both installed on your machine.
# Getting started and setup documentation
## Cloning the repo
```
git clone https://github.com/peedudu/lentokonepeli
```
## 
## Make sure you are in the root directory of the project. Before cding into one of the directories

### Python/Backend
```
cd backend
pip install -r requirements.txt
```
### Node/Frontend
```
cd lentokonepeli
npm install
```
# Running the app in development mode
## Backend and frontend in same terminal
```
cd lentokonepeli
npm run dev
```
## Backend and frontend in separate terminals
### Backend
```
cd backend
python ./backend_app.py
```
### Frontend
```
cd lentokonepeli
npm run dev:frontend
```
